# NEON DODGE: Hard Mode

A fast-paced, neon-style 2D arcade dodge game built with Python and Pygame. Survive the falling hazards, use your dash wisely, and collect power-ups to slow down time.

## Controls

* **Move:** Left / Right Arrows or **A / D**
* **Dash:** **Space** (short burst with cooldown and brief invulnerability)
* **Restart (after death):** **R**
* **Quit:** **Esc**

## Features

Slow-Mo: Collect yellow diamonds to slow down time.
Difficulty: Hazards speed up as your score increases.
High Score: Saves your best result automatically.

## Installation

Install the required library:
pip install -r requirements.txt

## Run the game

python main.py
