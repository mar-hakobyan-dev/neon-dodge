# NEON DODGE: Hard Mode

A fast-paced 2D dodge game made with Python and Pygame.

## Controls

* **Move:** Left / Right Arrow or **A / D**
* **Dash:** **Space** (short burst with cooldown and brief invulnerability)
* **Restart (after death):** **R**
* **Quit:** **Esc**

## Installation

Install the required library:

pip install -r requirements.txt

## Run the game

python main.py

## Technologies

* Python
* Pygame
